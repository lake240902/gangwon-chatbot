## 📌 프로젝트 개요

본 프로젝트는 강원대학교 데이터사이언스학과 석사과정 졸업 프로젝트로,  
김우주 교수님의 RAG 기반 강원도 관광 AI 컨시어지 챗봇을 기반으로  
**HCD(인간 중심 디자인) 방법론**을 적용하여 사용자 맞춤형 코스 추천 기능을 추가 구현하였습니다.

## 🆕 추가 구현 기능: 🗺️ 맞춤형 코스 추천

기존 5개 탭(AI 챗봇, 일정 생성기, TOP 추천, 비교 분석, 리뷰 통계)에  
**맞춤형 코스 추천 탭**을 추가 구현하였습니다.

### 핵심 특징
- ✅ **설문조사 32명 실제 데이터 기반** 추천
- ✅ **연령대** (20대 / 30대 / 40대 / 50대) 반영
- ✅ **동행 유형** (혼자 / 연인 / 친구 / 가족) 반영  
- ✅ **여행 성향** (관광 / 휴식 / 맛집 / 액티비티) 반영
- ✅ 설문 통계 수치 실시간 표시

### 추천 로직
1. 사용자가 연령대 · 동행 유형 · 여행 성향 선택
2. 설문 분석에서 도출된 그룹별 선호 키워드 매핑
3. 여행 성향에 맞는 춘천 지역 코스 4개 제시
4. 설문 통계 근거 함께 제공 (투명한 추천)

## ▶️ 실행 방법

### 1. 저장소 클론
```bash
git clone https://github.com/lake240902/gangwon-chatbot.git
cd gangwon-chatbot
```

### 2. 패키지 설치
```bash
pip install -r requirements.txt
```

### 3. API 키 설정
`.streamlit/secrets.toml` 파일 생성:
```toml
OPENAI_API_KEY = "your-api-key-here"
```

### 4. 실행
```bash
streamlit run app.py
```
